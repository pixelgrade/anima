# Post Titles Styling

In LT themes powered by Anima, post titles can be styled automatically to add an extra layer of typography without changing the title text itself. The feature applies Hive-inspired emphasis rules to supported post titles in single views and collection cards, so the same title can gain contrast and rhythm based on punctuation and letter case.

You can enable or disable the feature from `Appearance → Customize → Style Manager → Tweak Board` by toggling `Auto-style post titles`.

## Titles Styling Rules

When `Auto-style post titles` is enabled, the title treatment follows these rules:

- uppercase words, and words ending in an exclamation mark, are emphasized in bold;
- text inside quotes or parentheses is emphasized in italic;
- text between a `:` and a following `?` or `!` is emphasized in italic;
- text before a `:` is emphasized in bold;
- in titles with a single `?`, the text before the question mark is emphasized in bold and the text after it in italic.

Because the styling is generated automatically, you can write normal post titles and let the theme decide which fragments need extra emphasis.
